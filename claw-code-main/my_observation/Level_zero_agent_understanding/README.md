## 💡 Key Concepts

### Agent = Smart Employee
An agent is like an employee who:
- Has a list of tasks (commands)
- Has tools to do those tasks
- Knows when/how to use them
- Respects boundaries (permissions)
- Remembers what happened (memory)
- Works within budget

### Why CLAW-Code?
CLAW-Code is a Python rewrite of Claude Code's agent system, created after the source was exposed. It shows how modern agent harnesses work without using proprietary code.

### The Basic Flow
1. **Bootstrap** - Prepare systems
2. **Route** - Match request to tools
3. **Execute** - Run matched tools
4. **Respond** - Return results
5. **Persist** - Save state
